> 🖊️ منصة كتابة الروايات بالذكاء الاصطناعي — الكتابة، مساعدة الذكاء الاصطناعي، إدارة العالم، التسلسل التلقائي، ساحة النشر، ونظام الدفع

## ✨ الميزات الرئيسية

- **محرر غامر** — تخطيط من 3 أجزاء، وضع الآلة الكاتبة، وضع التركيز، الوضع الداكن
- **مساعد كتابة الذكاء الاصطناعي** — 18 أمرًا مُعدًا مسبقًا، إخراج متدفق، محرك ذاكرة النص الطويل
- **إدارة الفصول** — سحب وإفلات، سجل الإصدارات، بحث نصي كامل
- **التسلسل التلقائي** — عملية Bun مستقلة، توليد فصول مجدول بالذكاء الاصطناعي
- **مكتبة الإعدادات** — الشخصيات/العالم/العناصر، توليد تلقائي بالذكاء الاصطناعي
- **ساحة النشر** — نشر عام، صفحة قراءة، إعجابات/إشارات مرجعية
- **نظام الدفع** — دفع Alipay، نظام الرموز، أكواد الاسترداد
- **لوحة الإدارة** — 10 علامات تبويب، نسخ احتياطي للبيانات

## 🚀 البدء السريع

```bash
git clone https://github.com/your-username/moling-writing.git
cd moling-writing
bun install
cp .env.example .env
bun run db:push
bun run dev
```
