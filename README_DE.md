> 🖊️ KI-gestützte Roman-Schreibplattform — Schreiben, KI-Assistenz, Weltverwaltung, automatische Serialisierung, Veröffentlichungs-Plaza und Zahlungssystem

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

## ✨ Hauptmerkmale

- **Immersiver Editor** — 3-Spalten-Layout, Schreibmaschinenmodus, Fokusmodus, Dunkelmodus
- **KI-Schreibassistent** — 18 voreingestellte Befehle, Streaming-Ausgabe, Langtext-Gedächtnismaschine
- **Kapitelverwaltung** — Drag & Drop, Versionsverlauf, Volltextsuche
- **Automatische Serialisierung** — Unabhängiger Bun-Prozess, geplante KI-Kapitelgenerierung
- **Einstellungsbibliothek** — Charaktere/Welt/Items, KI-auto-generierung
- **Veröffentlichungs-Plaza** — Öffentliche Veröffentlichung, Leseseite, Likes/Lesezeichen
- **Zahlungssystem** — Alipay-Zahlung, Token-System, Einlösecodes
- **Admin-Dashboard** — 10 Tabs, Datensicherung

## 🚀 Schnellstart

```bash
git clone https://github.com/YinjiangCN/moling-writing.git
cd moling-writing
bun install
cp .env.example .env
bun run db:push
bun run dev
```

Admin-Konto: `admin@moli.com` / `admin123`
