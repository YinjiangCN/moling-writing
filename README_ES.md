[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

## ✨ Características Principales

- **Editor Inmersivo** — Diseño de 3 paneles, modo máquina de escribir, modo enfoque, modo oscuro
- **Asistente de Escritura IA** — 18 comandos preestablecidos, salida de transmisión, motor de memoria de texto largo
- **Gestión de Capítulos** — Arrastrar y soltar, historial de versiones, búsqueda de texto completo
- **Serialización Automática** — Proceso Bun independiente, generación programada de capítulos por IA
- **Biblioteca de Configuración** — Personajes/Mundo/Objetos, generación automática por IA
- **Plaza de Publicación** — Publicación pública, página de lectura, me gusta/marcadores
- **Sistema de Pago** — Pago Alipay, sistema de tokens, códigos de canje
- **Panel de Administración** — 10 pestañas, copia de seguridad de datos

## 🚀 Inicio Rápido

```bash
git clone https://github.com/your-username/moling-writing.git
cd moling-writing
bun install
cp .env.example .env
bun run db:push
bun run dev
```
