## ✨ Fonctionnalités Principales

- **Éditeur Immersif** — Disposition à 3 panneaux, mode machine à écrire, mode concentration, mode sombre
- **Assistant d'Écriture IA** — 18 commandes prédéfinies, sortie en flux, moteur de mémoire à texte long
- **Gestion des Chapitres** — Glisser-déposer, historique des versions, recherche plein texte
- **Sérialisation Automatique** — Processus Bun indépendant, génération planifiée de chapitres par IA
- **Bibliothèque de Paramètres** — Personnages/Monde/Objets, génération automatique par IA
- **Place de Publication** — Publication publique, page de lecture, likes/favoris
- **Système de Paiement** — Paiement Alipay, système de jetons, codes de remboursement
- **Tableau de Bord Admin** — 10 onglets, sauvegarde des données

## 🚀 Démarrage Rapide

```bash
git clone https://github.com/your-username/moling-writing.git
cd moling-writing
bun install
cp .env.example .env
bun run db:push
bun run dev
```

Compte administrateur : `admin@moli.com` / `admin123`
