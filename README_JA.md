> 🖊️ AI搭載のウェブ小説執筆プラットフォーム — 執筆、AIアシスタント、世界観管理、自動連載、作品広場、決済システムを統合

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

## ✨ 主な機能

- **没入型エディタ** — 3ペインレイアウト、タイプライターモード、集中モード、ダークモード
- **AI執筆アシスタント** — 18種類のプリセット、ストリーミング出力、長文脈記憶エンジン
- **章管理** — ドラッグ&ドロップ並び替え、バージョン履歴、全文検索
- **自動連載** — 独立Bunプロセスによる定期AI章生成
- **設定ライブラリ** — キャラクター/世界観/アイテム、AI自動生成
- **作品広場** — 公開出版、読者閲覧ページ、いいね/ブックマーク
- **決済システム** — Alipay決済、トークンポイント制、兑换コード
- **管理バックエンド** — 10タブ、データバックアップ

## 🚀 クイックスタート

```bash
git clone https://github.com/YinjiangCN/moling-writing.git
cd moling-writing
bun install
cp .env.example .env
bun run db:push
bun run dev
```
