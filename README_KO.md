## ✨ 주요 기능

- **몰입형 에디터** — 3패널 레이아웃, 타자기 모드, 집중 모드, 다크 모드
- **AI 작성 보조** — 18가지 프리셋, 스트리밍 출력, 장문 맥락 기억 엔진
- **챕터 관리** — 드래그 앤 드롭 정렬, 버전 히스토리, 전문 검색
- **자동 연재** — 독립 Bun 프로세스, 정기 AI 챕터 생성
- **설정 라이브러리** — 캐릭터/세계관/아이템, AI 자동 생성
- **작품 광장** — 공개 출판, 독자 열람 페이지, 좋아요/북마크
- **결제 시스템** — Alipay 결제, 토큰 포인트제, 교환 코드
- **관리 백엔드** — 10탭, 데이터 백업

## 🚀 빠른 시작

```bash
git clone https://github.com/your-username/moling-writing.git
cd moling-writing
bun install
cp .env.example .env
bun run db:push
bun run dev
```

관리자 계정: `admin@moli.com` / `admin123`
