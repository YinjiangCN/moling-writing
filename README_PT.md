[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

## ✨ Principais Recursos

- **Editor Imersivo** — Layout de 3 painéis, modo máquina de escrever, modo foco, modo escuro
- **Assistente de Escrita IA** — 18 comandos predefinidos, saída de streaming, motor de memória de texto longo
- **Gestão de Capítulos** — Arrastar e soltar, histórico de versões, pesquisa de texto completo
- **Serialização Automática** — Processo Bun independente, geração programada de capítulos por IA
- **Biblioteca de Configurações** — Personagens/Mundo/Itens, geração automática por IA
- **Praça de Publicação** — Publicação pública, página de leitura, curtidas/favoritos
- **Sistema de Pagamento** — Pagamento Alipay, sistema de tokens, códigos de resgate
- **Painel de Administração** — 10 abas, backup de dados

## 🚀 Início Rápido

```bash
git clone https://github.com/your-username/moling-writing.git
cd moling-writing
bun install
cp .env.example .env
bun run db:push
bun run dev
```
