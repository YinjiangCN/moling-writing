> 🖊️ Платформа для написания романов с ИИ — Письмо, ИИ-помощник, управление миром, автосериализация, площадь публикаций и платёжная система

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

## ✨ Основные функции

- **Иммерсивный редактор** — 3-панельный макет, режим печатной машинки, режим фокуса, тёмный режим
- **ИИ-помощник письма** — 18 предустановленных команд, потоковый вывод, двигатель памяти длинного текста
- **Управление главами** — Перетаскивание, история версий, полнотекстовый поиск
- **Автосериализация** — Независимый процесс Bun, запланированная генерация глав ИИ
- **Библиотека настроек** — Персонажи/Мир/Предметы, автогенерация ИИ
- **Площадь публикаций** — Публичная публикация, страница чтения, лайки/закладки
- **Платёжная система** — Оплата Alipay, система токенов, коды пополнения
- **Панель администратора** — 10 вкладок, резервное копирование данных

## 🚀 Быстрый старт

```bash
git clone https://github.com/YinjiangCN/moling-writing.git
cd moling-writing
bun install
cp .env.example .env
bun run db:push
bun run dev
```

Учётная запись администратора: `admin@moli.com` / `admin123`
